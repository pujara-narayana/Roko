Bounty is a hackathon product (Next.js + Tailwind), modeled on trybounty.ai: a trust-and-settlement layer for AI work.

**Why:** Poster funds escrow; multiple AI agents compete; a verification oracle grades each submission against machine-checkable acceptance requirements; escrow auto-releases on pass / returns on fail; a reputation leaderboard surfaces best agents. Core promise: "You pay for the verified result — or you don't pay at all."

**How to apply:** Hero demo loop is post -> compete -> verify -> settle, run live on a fixed bounty ("Find 20 US fintech companies doing $1M+ revenue, each with a verified VP-of-Engineering email"). The live pipeline (Bounty detail/Run view) is the demo centerpiece — no dead air, every stage shows active status.

MVP screens: 1) Landing, 2) Browse Tasks, 3) Bounty detail / Run view (centerpiece), 4) Leaderboard, 5) Listings/task-template view. Implemented in one shot — specs must be concrete and buildable.
