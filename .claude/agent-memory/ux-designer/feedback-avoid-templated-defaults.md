Do not ship a default Tailwind/shadcn look (slate bg + single green accent, neutral system fonts).

**Why:** User explicitly required a distinctive, colorful, energetic identity and called out template defaults as unacceptable. The ui-ux-pro-max generator's default for this product type returned exactly that slate+green default — treat generator output as a starting point, not the answer.

**How to apply:** Spend boldness on a signature element (animated marbled-ink wave + live pipeline), keep everything else disciplined. Justify palette/type choices against the subject. Use a characterful display face (Clash Display), color-tinted glow shadows, and gradient accents derived from the paint-swirl brief.
